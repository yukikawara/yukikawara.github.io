# yukikawara.github.io

Jekyll と GitHub Pages を使用した個人サイト + ブログ

## 概要
このリポジトリは Yuki Kawara の個人サイトとブログです。

## ローカル開発

1. 依存関係をインストール:
```bash
bundle install
```

2. ローカルサーバーを起動:
```bash
bundle exec jekyll serve
```

3. ブラウザで `http://localhost:4000` を開く

## ブログ記事の追加
`_posts/` ディレクトリに `YYYY-MM-DD-記事タイトル.md` 形式でMarkdownファイルを追加してください。

## デプロイ
mainブランチにプッシュすると自動的にGitHub Pagesにデプロイされます。
